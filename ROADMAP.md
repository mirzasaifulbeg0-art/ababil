# ABABIL — Project Roadmap

**Tagline:** Nature • Technology • Trust
**Theme:** Modern, Minimal, Premium — Green / White / Navy Blue

## What we're building
One premium website containing 4 businesses:
1. Digital Services (application assistance — NOT live government integrations)
2. Natural Products (eCommerce: honey, organic goods, etc.)
3. Islamic Library (free / public-domain PDF books only)
4. Blog (SEO content)
Plus: user accounts, admin panel, payments.

## Important realities (design around these)
1. Digital services = "application assistance" leads. We do not connect to gov portals.
2. Payments require real KYC business accounts. We build the code; keys plugged in later. Test mode until then.
3. Islamic Library holds only legally free / public-domain PDFs, uploaded by the owner.

## Build order (approval-gated)
1. Roadmap  ✅ (this file)

13. SEO + Performance + Security pass
14. Deploy to Vercel

## Tech stack
- Frontend: Next.js (App Router), React, TypeScript, Tailwind CSS
- Backend: Next.js API Routes / Server Actions
- Database: PostgreSQL + Prisma ORM
- Auth: NextAuth
- Storage: Cloudinary
- Deploy: Vercel

## Rule
Build one module at a time. Wait for owner approval before moving to the next.
